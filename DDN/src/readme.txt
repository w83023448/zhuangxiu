1、包结构:
	
	base						基础性功能模块
	ddn							ddn项目相关模块
	   exception				异常模块
	   manager					管理类
	   mudle					功能模块包
	      base					基础模块,如:用户管理
	      contacts				联系相关,如:留言等
	   rpc						对外接口(所有ajax对外请求)
	   util						项目内部使用的工具类包
	----------------------------------------------------	   
约束: 所有ajax请求,均只能从rpc中查找.
